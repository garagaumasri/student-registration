# Student Registration System

## 📌 Project Description
This is a simple Student Registration System built using PHP and MySQL.  
It allows users to register students and store data in a database.

---

## 💻 Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP
- Git & GitHub

---

## ⚙️ Features
- Student Registration Form
- Save data into database
- View student records
- Git version control

---

## 🚀 How to Run Project

1. Start XAMPP (Apache & MySQL)
2. Place project in `htdocs` folder
3. Import database in phpMyAdmin
4. Open browser:
   http://localhost/student-registration/

---

## 📂 Project Structure
- index.php (Registration Form)
- insert.php (Insert Data)
- db.php (Database Connection)
- view.php (Display Data)
